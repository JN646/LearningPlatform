Limiter

Cubase has a built in limiter to get you started with your dynamic processing either while mixing or mastering.

We can access this by adding the Limiter plugin from the Dynamics insert menu to a blank insert slot on a channel.

By setting the Output volume the gain reduction will prevent the signal hitting that value. Excessive limiting will result in a pumping effect. By changing the Release time we can choose how quickly the Gain Reduction will take place.

See the Limiter section for more information on how to use this.
