Backing up your Project

Ensuring your project is kept safe is paramount in a professional workplace. All of the hard work putting into creating and recording a piece of music could be lost if your work is not stored appropriatly.

Cubase has a feature that allows you to export your whole project with all included files to a new location.

From the File menu, you can choose the Back up Project... option.

This will open a window that will allow you to select a new project folder.

This is also useful if you wish to transfer your work from one computer to another. By just copying the .CPR folder you are not ensuring that all attatched media is copied as well. So you could find that you are missing audio and settings when you open the file on another machine.
